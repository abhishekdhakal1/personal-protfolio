# 🚀 My Personal Website

Welcome to the source code of my personal portfolio. This project is built to be fast, bold, and visually striking, featuring custom React components and high-energy Tailwind CSS animations.

---

## 🛠 Tech Stack

| Technology       | Use Case                  |
| :--------------- | :------------------------ |
| **React.js**     | Frontend Framework        |
| **Tailwind CSS** | Utility-first Styling     |
| **Vite**         | Build Tool (Fast Refresh) |
| **Lucide React** | Iconography               |

---

## ✨ Key Features

- **Aggressive Home Animation:** A custom-built `HomeSection` that renders 50+ animated text instances with randomized positioning and staggered fade-in/out effects.
- **Modern UI/UX:** Clean, dark-themed aesthetic with high-contrast typography.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views using Tailwind's breakpoint system.
- **High Performance:** Minimal external library dependencies to ensure lightning-fast load times.
